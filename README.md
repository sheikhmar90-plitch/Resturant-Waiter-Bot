# 🍽️ Restaurant Waiter Chatbot

## Overview

The Restaurant Waiter Chatbot is a Python-based restaurant ordering system that allows users to view menu items, explore special offers, place orders, track quantities, and generate a final bill. The chatbot uses fuzzy string matching to handle spelling mistakes and follows strict menu constraints to ensure reliable and accurate responses.

## Features

- Display restaurant menu and prices
- Show special deals and offers
- Place orders with quantity selection
- Smart typo correction using fuzzy matching
- Input validation and normalization
- Order tracking using dictionaries
- Automatic bill generation
- Discount calculation for eligible orders
- Restaurant-branded receipt generation
- Strict menu-based chatbot behavior

## Technologies Used

- Python
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Difflib (Fuzzy Matching)

## How It Works

1. The chatbot welcomes the customer.
2. Users can view the menu or special offers.
3. Customers select items and quantities.
4. The system validates and stores orders.
5. Fuzzy matching handles minor spelling mistakes.
6. The bill is calculated based on quantity and price.
7. Discounts are applied when conditions are met.
8. A final receipt is generated for the customer.
